# Modulo 3 / Unidad 3

### Esta actividad se divide en 4 partes:
- Realice un programa en JS que cree el Object Weather, deberá tener las
siguientes propiedades: Temperatura mínima, temperatura máxima, clima (es
decir: soleado, parcialmente nublado, nublado, lluvia o nieve), velocidad del
viento.
- Añadir al Object Weather el método: Temperatura media del día.
- Escriba un programa que guarde en un array unidimensional los datos de la
temperatura de una semana y calcule la media de las temperaturas máximas y
la media de las temperaturas mínimas.
- Utilizando DOM, realice una web que muestre un pronóstico de 7 días

Link to demo: https://modulo-3-unidad-3.vercel.app
